# Restaurant Menu & Order Management System (Python)

This project is developed as a Final Project for Programming for Business (PF Lab).

## Project Overview
A command-line based Restaurant Menu & Order Management System that allows restaurants to manage dishes, take orders, track inventory, generate bills, and analyze sales.

## Features
- Add, update, remove, and search menu items
- Order processing with table management
- Automated bill generation with tax, discounts, and promotions
- Buy 1 Get 1 Free offer on appetizers
- Seasonal discount on desserts
- Ingredient inventory tracking with low-stock alerts
- Sales and inventory reports
- Basic forecasting based on past order data

## Technologies Used
- Python 3
- Dictionaries, Lists, Functions
- Command Line Interface (CLI)
